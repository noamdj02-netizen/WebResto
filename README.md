- **💬 Support 24/7** - Accompagnement personnalisé

## 🚀 Installation et Démarrage

### Prérequis
- Serveur web (Apache, Nginx, ou serveur Node.js)
- Navigateur web moderne

### Installation Rapide
1. Clonez le repository :
```bash
git clone https://github.com/votre-username/webresto.git
cd webresto
```

2. Démarrez le serveur Node.js :
```bash
node server.js
```

3. Ouvrez votre navigateur et allez sur `http://localhost:3000`

### Installation sur Serveur Web
1. Uploadez tous les fichiers sur votre serveur web
2. Assurez-vous que votre serveur supporte les fichiers statiques
3. Configurez votre domaine pour pointer vers le dossier du projet

## 🎨 Personnalisation

### Couleurs
Les couleurs principales peuvent être modifiées dans `style.css` :
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #27ae60;
    --text-color: #2c3e50;
}
```

### Logo et Images
- Remplacez l'emoji 🍽️ par votre logo
- Ajoutez vos propres images dans le dossier `images/`
- Personnalisez les icônes et emojis dans le HTML

### Contenu
- Modifiez les textes dans les fichiers HTML
- Ajustez les prix dans `script.js`
- Personnalisez les fonctionnalités selon vos besoins

## 🔧 Configuration Avancée

### Variables d'Environnement
Créez un fichier `.env` pour configurer :
```env
PORT=3000
NODE_ENV=production
```

Pour une version production, considérez l'ajout d'une base de données :

- PostgreSQL pour les commandes
